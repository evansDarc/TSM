# TSM

# Tunel SSH Manager

Aplicación de escritorio para gestionar túneles SSH a MySQL de forma simple.

## Características
- Interfaz gráfica JavaFX
- No requiere Java instalado
- Runtime optimizado con jlink
- Windows x64
